SimpleSort Bookmarks

Backup your bookmars

This addon will sort every folder not in the blacklist. If you dont like the new order the only way to revert to the previous situation is to restore from your backup. There isn't an undo feature in this addon and the backup is only two clicks away.

I warmly recommend to save you bookmarks before proceeding.

Blacklist of bookmark folders

The folders in this list and their children will not be sorted.

    This is usefull if you have a folder where you put temporarly links you prefer to be left sorted by creation time.

    Sort all bookmarks

    If you have a LOT of bookmarks it may take a while to sort everything.

    More options are available through the Add-ons Manager interface.
